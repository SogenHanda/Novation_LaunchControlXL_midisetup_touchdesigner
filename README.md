This is a .tox file for setting Novation LaunchControl XL useing in TouchDesigner.
If you use launchcontrol xl in touchdesigner, you can use this tox file.


This tox file enables the following:

・Simplified MIDI channel naming during MIDI controller setup

・Support for both MIDI controller operation and GUI operation

・Ability to build MIDI controller mappings without connecting a MIDI controller (select MIDI = None)
